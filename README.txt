Number Clicker
Number Clicker is a simple and modern app with a clean GUI for counting clicks. It lets you track your clicks, reset your score, and switch between light and dark mode effortlessly.

Main features include a big rounded click button, a reset button, automatic score saving, and a toggle to switch between light and dark themes. The design is sleek and user-friendly for a smooth experience.

How to use
Just run Number_Clicker.exe and start clicking the big rounded button to increase your count. Use the reset button to set your score back to zero. The toggle button allows you to switch between light and dark mode anytime you want.
